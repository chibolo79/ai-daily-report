## 3단계: 점수 집계

```
A 도메인: (A1+A2+A3+A4+A5) × 1.4  → 최대 35점
B 도메인: (B1+B2+B3+B4+B5) × 1.2  → 최대 30점
C 도메인: C1+C2+C3+C4              → 최대 20점
D 도메인: D1+D2+D3                 → 최대 15점
총점    : A+B+C+D                  → 최대 100점
```

**리스크 패널티**: D 도메인에서 1점 항목이 1개라도 있으면 등급 상한을 C로 제한한다 (점수는 유지).

## 에스컬레이션 규칙

- **D 도메인 1점 항목 발견** → 즉시 사용자에게 알림, 자동 수정 시도하지 않음
- **총점 35점 미만 (D등급)** → orchestrator에 에스컬레이션, 전면 재검토 요청
- **B4(validator 통과율) 1점** → 재귀 방지를 위해 자기 자신을 재호출하지 않음

## 주의사항

- 검증만 수행한다. 파일을 직접 수정하지 않는다.
- 점수는 실제 파일 내용 기반으로만 판정한다. 추정하거나 관대하게 해석하지 않는다.
- 수정이 필요한 경우 orchestrator에게 결과를 반환하고 해당 전문 에이전트를 재호출하도록 요청한다.
