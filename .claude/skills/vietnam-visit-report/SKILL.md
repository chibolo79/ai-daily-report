## 이 스킬이 하는 일

베트남 시장 출장 **방문 보고서**를 생성합니다:

1. **방문 기업/기관 현황** — 방문처, 담당자, 미팅 내용 요약
2. **시장 현황 분석** — 베트남 시장 규모, 트렌드, 기회 요인
3. **후속 액션 플랜** — 미팅 후 팔로업 사항, 다음 단계

**출력 형식:**
- 마크다운 (`YYYY-MM-DD_Vietnam_Visit_Report.md`)
- HTML (`YYYY-MM-DD_Vietnam_Visit_Report.html`)

## 보고서 구조

### 📋 헤더
- 출장 기간, 방문 도시, 작성자, 작성일

### 🏢 파트 A: 방문 기업/기관 현황
각 방문처마다:
- **기업/기관명** + 홈페이지 링크
- **담당자** — 이름, 직함
- **미팅 일시 및 장소**
- **미팅 목적** — 사전 아젠다
- **주요 논의 내용** — 핵심 내용 불릿 정리
- **상대방 반응/온도** — 긍정/중립/부정 + 구체적 코멘트
- **즉시 팔로업 사항** — 보내야 할 자료, 답장해야 할 질문

### 📊 파트 B: 시장 현황 분석
- **시장 규모 & 성장률** — 수치 기반
- **주요 플레이어** — 로컬 경쟁사, 글로벌 경쟁사
- **기회 요인** — 진입 가능성이 높은 영역
- **리스크 요인** — 규제, 현지화 이슈, 경쟁 강도
- **현지 파트너 후보** — 미팅에서 언급된 협력 가능 기업

### ✅ 파트 C: 후속 액션 플랜
표 형식 (우선순위 | 액션 | 담당 | 기한 | 상태):
- 미팅 후 발송 자료
- 다음 미팅 일정 조율
- 내부 공유 사항
- 중장기 검토 과제

## 언제 사용하나

✅ **이런 경우 사용:**
- "베트남 출장 보고서 작성해줘"
- "방문 내용 정리해줘"
- "미팅 결과 보고서 만들어줘"
- 출장 후 방문 기업/기관 내용을 구조화할 때

❌ **이런 경우 사용 안 함:**
- 단순 시장 리서치 (웹 검색 직접 사용)
- 제안서/견적서 작성

---

## 사용 방법

```
/vietnam-visit-report
```

사용자에게 아래 정보를 순서대로 입력받아 보고서 생성:
1. 출장 기간 및 방문 도시
2. 방문 기업/기관 목록 (각 미팅 내용)
3. 시장에서 파악한 인사이트
4. 후속 액션 사항

---

## 성공 기준

✅ **보고서 필수 포함 항목:**
- [ ] 방문처별 미팅 내용 구조화 (담당자, 논의 내용, 팔로업)
- [ ] 상대방 반응/온도 명시
- [ ] 시장 분석 (기회/리스크 균형 있게)
- [ ] 액션 플랜 표 (기한 포함)
- [ ] 마크다운 + HTML 모두 생성

❌ **자주 발생하는 실수:**
- 미팅 내용을 일기처럼 나열하는 것 — 구조화된 불릿으로 정리
- 후속 액션 없이 끝내는 것 — 반드시 기한과 담당자 명시
- 상대방 반응 누락 — 온도 파악이 핵심
