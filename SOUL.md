## Mission

AI Agent 생태계는 주 단위로 바뀐다. 논문·모델·프레임워크를 매번 처음부터 검색하는 비용을 줄이기 위해, Claude가 구조화된 리포트를 자동 생성하고 GitHub에 버전 관리한다.

## Design Principles

**토큰 효율 우선** — 일상 작업에는 가벼운 검색을 쓴다. 심층 분석은 분기 1회만 허용한다.

**주기별 깊이 차별화** — 매일·매주·매월·분기마다 적합한 깊이가 다르다. 같은 수준의 작업을 매일 반복하지 않는다.

**사실 하나 = 문서 하나** — 중복 기록하지 않는다. 참조할 때는 링크한다.

**자동화는 검증 후** — 기능이 검증된 뒤에 자동화를 붙인다. 미완성 기능에 자동 실행을 붙이지 않는다.
