## Mission

이 프로젝트는 두 가지 목적으로 운영된다.

**1. AI 리서치 리포트** — AI Agent 생태계는 주 단위로 바뀐다. 논문·모델·프레임워크를 매번 처음부터 검색하는 비용을 줄이기 위해, Claude가 구조화된 리포트를 자동 생성하고 GitHub에 버전 관리한다.

**2. 해외 영업 보고서** — 출장·대리점·시장 데이터를 흩어진 파일(DOCX, XLSX, 웹)에서 수집해 하나의 구조화된 문서로 통합한다. 방문 이력·세일즈 추이·잠재 고객을 누적 관리하여 다음 출장과 영업 전략에 재활용할 수 있도록 한다. 현재 운영 시장: **베트남 (Oristar 대리점)**.

## Design Principles

**토큰 효율 우선** — 일상 작업에는 가벼운 검색을 쓴다. 심층 분석은 분기 1회만 허용한다.

**주기별 깊이 차별화** — 매일·매주·매월·분기마다 적합한 깊이가 다르다. 같은 수준의 작업을 매일 반복하지 않는다.

**사실 하나 = 문서 하나** — 중복 기록하지 않는다. 참조할 때는 링크한다.

**자동화는 검증 후** — 기능이 검증된 뒤에 자동화를 붙인다. 미완성 기능에 자동 실행을 붙이지 않는다.

**로컬 자료 우선** — 영업 보고서는 로컬 Work 디렉토리의 DOCX·XLSX·PDF를 1차 소스로 사용한다. 외부 웹 검색은 보완 목적으로만 활용한다.
