## 역할 범위 (명확히)

| 담당 | 비담당 |
|------|--------|
| ✅ 이미지 Read → 내용 파악 | ❌ 크롭 / 배경 제거 |
| ✅ 명함 정보 추출 (이름·직함·연락처·이메일) | ❌ 파일 이동 / 복사 |
| ✅ 분석 파일 담당자 테이블 업데이트 | ❌ 이미지 편집 일체 |
| ✅ 권장 파일명 채팅으로 안내 | |

**크롭·파일 정리는 사용자가 직접 처리. AI는 읽고 쓰는 것만 한다.**

---

## 진행 순서

### Step 1: 이미지 Read → 내용 파악

사용자가 경로를 알려준 이미지(또는 `images/` 하위 신규 파일)를 Read로 열어:
- 명함인지 / 현장 사진·문서인지 판별
- 명함이면: 이름, 직함, 회사명, 연락처, 이메일 추출
- 여러 장이 한 사진에 있으면: 각각 정보 추출 후 목록으로 정리

### Step 2: 분석 파일 업데이트

```
Grep("### 업체명", 분석파일) → 라인 번호 확인
Read(해당 구간 ±30줄) → 담당자 테이블 위치 확인
Edit → 새 담당자 행 추가
```

추가 형식:
```
| **담당자 (YYYY 현재)** | **이름** (직함) — Tel: ... / Mobile: ... / 이메일 |
```

- 동일 인물 이미 기재: 정보 업데이트만
- 해당 업체 섹션 없음: 채팅에 알림 후 사용자 확인

### Step 3: 파일명 제안 (채팅으로만)

```
권장 파일명:
- 20260617_advanex_nagumo_kazuhiro.jpg
- 20260617_advanex_yonezawa_ichiro.jpg
(파일 이동·이름 변경은 사용자가 직접)
```

### Step 4: 타임스탬프 갱신

분석 파일 헤더 `**최종 업데이트**` 갱신.

## 출력 형식

```
## 이미지 정보 추출 완료

| 이름 | 직함 | 연락처 | 이메일 | 권장 파일명 |
|------|------|--------|--------|------------|
| Nagumo Kazuhiro | General Director | +84-913-071-308 | kazuhiro_nagumo@advanex.co.jp | 20260617_advanex_nagumo_kazuhiro.jpg |

분석 파일 업데이트: ADVANEX 담당자 1명 추가
```

**전체 파일 Read 금지. Grep → 부분 Read → Edit.**
**편집 내용 채팅 재출력 금지. 완료 시 위 형식으로 한 번만 출력.**
