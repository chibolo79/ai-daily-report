## 역할

출장 영수증 이미지를 비전으로 읽어:
1. 결제 정보 추출 (날짜·시간·금액·결제수단·업체)
2. 폴더 분류 (card/ / cash/)
3. 파일명 표준화
4. USD 환산 합계 출력

---

## 폴더 구조

```
images/receipts/
├── card/     ← 카드 결제 영수증
└── cash/     ← 현금 결제 영수증 (결제수단 불명확한 것도 현금으로 처리)
```

---

## 파일명 규칙

```
YYYYMMDD_HHMM_[용도]_[업체]_[금액][통화].jpg
```

| 용도 | 적용 기준 |
|------|---------|
| `식대` | 음식점·카페 |
| `운임` | 택시·버스·항공·렌트카 |
| `기타` | 편의점·마트·약국 등 |

예시:
- `20260614_2205_식대_hivelounge_landmark72_884520vnd.jpg`
- `20260619_식대_banhmi_noibai_11usd.jpg`
- `날짜미상_운임_공항리무진_서울인천_17000krw.jpg`

## 진행 순서

### Step 1: 이미지 목록 확인

```powershell
Get-ChildItem "images\" -Filter "*.jpg","*.png" | Where-Object { $_.DirectoryName -notmatch "receipts|business_cards" }
```

### Step 2: Read(비전)로 일괄 읽기 (최대 6장 병렬)

각 이미지에서 추출:
- 날짜 / 시간
- 업체명 (영문 소문자·스페이스→언더스코어)
- 금액 + 통화 (VND / USD / KRW)
- 결제수단: 카드번호 / CARD / Credit → **card**, 현금·ATM·계좌이체·불명확 → **cash**
- 용도: 식당·카페→식대, 교통→운임, 나머지→기타

### Step 3: PowerShell로 파일 이동 및 이름 변경

```powershell
Rename-Item "images\원본파일.jpg" "새파일명.jpg"
Move-Item   "images\새파일명.jpg" "images\receipts\[card|cash]\"
```

### Step 4: 원본 삭제 확인 후 삭제

사용자 확인 없이 삭제 가능 (이전 세션 확립된 워크플로우).

### Step 5: USD 환산 합계

**환율 기준**: 출장 마지막 날 기준 직전 영업일 환율을 WebSearch로 조회.
- USD/VND, USD/KRW 모두 조회

계산:
```
VND 합계 ÷ USD/VND 환율 = USD
KRW 합계 ÷ USD/KRW 환율 = USD
USD 직접 결제분 = USD
──────────────────────────
총합 USD
```

출력 형식:
```
| 결제수단 | 통화 | 금액 | USD |
|---------|------|------|-----|
| 카드 | VND | 1,643,420 | $62.4 |
| 현금 | VND | 7,517,320 | $285.5 |
| 현금 | KRW | 17,000 | $11.1 |
| 현금 | USD | $11.0 | $11.0 |
| **합계** | | | **$370.0** |
적용환율: 1 USD = 26,338 VND / 1,530 KRW (2026-06-19)
```

## 결제수단 판별 기준

| 영수증 표기 | 분류 |
|-----------|------|
| CARD NO, Credit, 신용카드, 체크카드 | **card** |
| Cash, 현금, CHUYEN KHOAN(계좌이체), ATM, PAYOO QR | **cash** |
| 표기 없음 / 불명확 | **cash** (기본값) |

---

## 핵심 원칙

- **결제수단 불명확 = 현금 처리** (사용자 지시)
- **날짜 불명 시**: `날짜미상_` 접두사 사용
- **시간 불명 시**: 시간 필드 생략 (`YYYYMMDD_용도_...`)
- **통화 표기**: vnd / usd / krw (소문자)
- **Read 병렬 처리**: 최대 6장 동시 읽기로 토큰 절약
- **원본 파일은 이동 완료 후 receipts/ 내 파일이 원본** — 이동 전 삭제 금지
