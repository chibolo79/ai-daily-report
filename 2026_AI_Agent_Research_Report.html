파트 D: 업계 트렌드

1

추론 모델이 에이전트의 기본값

OpenAI o3 ARC-AGI 87.5% 달성(인간 85% 초과). 추론 vs 일반 모델 격차가 에이전트 선택 핵심 기준. Anthropic Mythos 5로 추론 경쟁 전면전.

2

Computer Use 경쟁 가속

Anthropic·OpenAI·Google 모두 GUI 직접 제어 출시. 액션 도구 비율 27% → 65%(16개월). 2027년 주류화 전망.

3

MCP 표준화 반등

초기 비판 후 2026년 중반 재급증. Firecrawl MCP 사용량 최근 한달 35% 성장. 네이티브 OAuth가 CLI 대비 차별화 강점.

4

멀티에이전트 시스템으로 구조 전환

단일 에이전트 → 전문화 에이전트 팀 + 오케스트레이터. Claude Code 서브에이전트, OpenAI Responses API가 뒷받침.

5

신뢰성·검증이 연구 무게 중심

MedAgentGym, WebDevJudge 등 평가·안전성 논문이 2026년 상위 컨퍼런스 점령. 성능 → 신뢰성으로 이동.

6

모델 패밀리 계층화 가속

Anthropic Mythos, OpenAI o3-pro 등 최상위 전용 티어 생성. "성능 vs 비용" → "작업 유형별 최적 모델" 매트릭스로 복잡화.
