1. Externalization in LLM Agents — 에이전트 메모리·스킬·프로토콜 총정리
📝 한 줄 요약: 에이전트가 "외부 세계"와 어떻게 상호작용하는지를 하나의 통일된 프레임워크로 정리한 서베이. (2026년 4월)
왜 중요한가: 에이전트 설계 시 기억 저장·툴 호출 구조에 대한 이론적 기반. Claude Code의 Agent Skills, MCP 등 실제 구현과 직결된다.
2. MAGMA — 다중 그래프 기반 에이전트 메모리 아키텍처
📝 한 줄 요약: 에이전트가 장기 기억을 구조화된 그래프로 관리해 장기 작업에서 맥락을 잃지 않게 하는 방법. (2026년)
왜 중요한가: "에이전트가 긴 작업 중에 앞에 한 것을 까먹는다"는 가장 흔한 실패 원인을 직접 해결한다. 프로덕션 에이전트 구축 시 참고 가치 높음.
3. How are AI agents used? — MCP 툴 17만 개 분석
📝 한 줄 요약: 실제로 사람들이 AI 에이전트에 어떤 툴을 붙여 쓰는지를 177,000개 데이터로 분석한 실증 연구. (2026년 3월)
왜 중요한가: 이론이 아닌 실제 사용 데이터. 어떤 에이전트 기능이 현업에서 많이 쓰이는지 파악하는 데 유용.
