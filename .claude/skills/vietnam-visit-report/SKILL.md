## 이 스킬이 하는 일

베트남 시장 출장 **방문 보고서**를 생성합니다:

1. **방문 기업/기관 현황** — 방문처, 담당자, 미팅 내용 요약
2. **시장 현황 분석** — 베트남 시장 규모, 트렌드, 기회 요인
3. **후속 액션 플랜** — 미팅 후 팔로업 사항, 다음 단계

**출력 형식:**
- 마크다운 (`YYYY-MM-DD_Vietnam_Visit_Report.md`)
- HTML (`YYYY-MM-DD_Vietnam_Visit_Report.html`)

## 보고서 구조

### 📋 헤더
- 출장 기간, 방문 도시, 작성자, 작성일

### 🏢 파트 A: 방문 기업/기관 현황
각 방문처마다:
- **기업/기관명** + 홈페이지 링크
- **담당자** — 이름, 직함
- **미팅 일시 및 장소**
- **미팅 목적** — 사전 아젠다
- **주요 논의 내용** — 핵심 내용 불릿 정리
- **상대방 반응/온도** — 긍정/중립/부정 + 구체적 코멘트
- **즉시 팔로업 사항** — 보내야 할 자료, 답장해야 할 질문

### 📊 파트 B: 시장 현황 분석
- **시장 규모 & 성장률** — 수치 기반
- **주요 플레이어** — 로컬 경쟁사, 글로벌 경쟁사
- **기회 요인** — 진입 가능성이 높은 영역
- **리스크 요인** — 규제, 현지화 이슈, 경쟁 강도
- **현지 파트너 후보** — 미팅에서 언급된 협력 가능 기업

### ✅ 파트 C: 후속 액션 플랜
표 형식 (우선순위 | 액션 | 담당 | 기한 | 상태):
- 미팅 후 발송 자료
- 다음 미팅 일정 조율
- 내부 공유 사항
- 중장기 검토 과제

사용자에게 아래 정보를 순서대로 입력받아 보고서 생성:

## 분석 파일 업데이트 절차

출장 후 미팅 내용을 `2026-06-14_Oristar_Vietnam_Sales_Analysis.md`에 누적할 때 아래 순서를 따른다.

### 편집 순서 (토큰 최소화)

1. **Grep** — 대상 회사명으로 라인 번호 확인
2. **Read(offset, limit)** — 해당 섹션 ±40줄만 읽음. 전체 파일 Read 금지
3. **Edit** — 변경 내용만 교체. Write(전체 재작성) 사용 금지
4. **채팅 재출력 없음** — 완료 후 한 줄로 종료

### 업체 정보 업데이트 입력 형식

사용자가 아래 형식으로 입력하면 파싱 없이 즉시 Edit 실행:

```
회사명 / 변경 항목 / 새 내용
예: CAMEX / 담당자 / Mr. A (Director)
    MATEX / 방문 이력 / 2026-XX-XX: ...
```

### 방문 이력 추가 형식

```markdown
- **YYYY년 MM월 방문 (N차)**: 논의 내용 핵심 불릿
```

### 새 업체 섹션 삽입

1. Grep(`---`)으로 삽입 위치(직전 섹션 구분선) 라인 번호 확인
2. Edit으로 해당 위치에 표준 포맷 섹션 삽입

### 표준 섹션 포맷

```markdown
### [회사명]
| 항목 | 내용 |
|------|------|
| **담당자** | ... |
| **주소** | ... |
| **연락처** | ... |
| **이메일** | ... |
| **홈페이지** | ... |

**회사 개요**: ...
**세일즈 현황**: ...
**방문 이력**: ...
**영업 포인트**: ...

---
```

- 미확인 항목: `(미확인 — 방문 시 확인 필요)` 로 명시. 공란 불가.
